# Typhoon Demo MP3s

Local MP3 demo recordings used by the public frontend (`/musik` page and the homepage hero player).

## Expected filenames

```
sen-benim.mp3
karanfill.mp3
gece-yine-dustun.mp3
farksilin.mp3
cilgin.mp3
bir-tek-sen.mp3
```

These are demo recordings — not final releases. No download button is exposed.
The audio player handles missing files gracefully and falls back to a static UI.

Do not commit raw / non-MP3 audio files (WAV, AIFF, FLAC, M4A, DAW project files).
