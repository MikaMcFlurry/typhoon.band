# Typhoon Design System

## Goal

Premium cinematic band website.

The website must feel:
- dark
- warm
- smoky
- musical
- elegant
- slightly raw
- high quality
- concert-poster-like

Not:
- generic
- yellow
- boxy
- cheap grunge
- template-like

## Color palette

Use refined gold/champagne/bronze tones.

Recommended palette:

```text
Deep black: #030201
Brown black: #060403
Warm dark brown: #100a06
Dark bronze: #6f4a1f
Deep gold: #b8873b
Antique gold: #c79a4b
Champagne gold: #e8c982
Warm cream: #f2e6cf
Muted text: #b9aa90
```

## Gold rules

Gold should:
- look like antique gold or champagne
- be used as an accent
- appear warm and premium
- work with sepia/dark backgrounds

Avoid:
- bright yellow
- neon yellow
- flat orange
- large flat yellow surfaces
- cheap glowing borders

## UI shape

Use smoother UI:
- card radius: around 18–28px
- pill buttons
- soft borders
- subtle inner highlights
- polished bronze/glass panels
- refined shadows

Avoid:
- sharp rectangular boxes everywhere
- harsh 1px borders on all elements
- generic card grids
- boxy template feeling

## Backgrounds

Use:
- black/dark brown gradients
- sepia glows
- subtle smoky overlays
- subtle grain

Avoid:
- heavy noisy texture
- low-contrast muddy text
- overused glow

## Typography

No external fonts in current phase.

Use system fonts with:
- clear hierarchy
- uppercase labels with tracking
- strong clean display text
- readable body text

Do not use cheap-looking Impact-style typography as the main design identity.

The handwritten Typhoon logo is the decorative script element.

## Buttons

Buttons should feel premium:
- pill shape
- gold/champagne accent
- clear hover states
- not too bright
- no harsh yellow

Recommended variants:
- primary gold/champagne
- secondary transparent dark with gold border
- quiet text link

## Cards / panels

Cards should feel like:
- premium glass/bronze panels
- soft dark surfaces
- slightly rounded
- subtle gold edge
- cinematic UI

Avoid:
- standard Bootstrap-style cards
- square boxes
- aggressive outlines

## Hero section

Hero must be the strongest visual element.

Required layer order:
1. hero image
2. dark/sepia gradients
3. content/text
4. demo player / feature UI
5. handwritten Typhoon logo overlay as visible top design layer

The hero logo must not be buried under gradients.

## Responsive rules

Desktop:
- cinematic, wide hero
- strong logo overlay
- compact premium feature cards

Tablet:
- reduce logo size
- keep hero image strong
- avoid cramped text

Mobile:
- no horizontal overflow
- readable text
- logo overlay smaller and repositioned
- buttons stack or wrap cleanly
- audio player fits width
